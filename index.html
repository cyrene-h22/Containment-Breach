Site Director's Notes

SCP-9328, code name Containment Breach, is a visual novel styled game that has the player work as a researcher of the SCP Foundation. The game lets the player experience it is like working at Site-28, lots of fun. The O5 Council has requested for SCP-9328 to be wiped from any devices not within the Foundation's network, but it'd always appear back on the desktops.
